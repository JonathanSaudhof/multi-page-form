# Buena multi-page-form

## Getting Started

### Setup

Run npm install first

```bash
pnpm i
```

### Development server

Run development server

```bash
pnpm dev
```

### Start production build

Run and start production build

```bash
pnpm dev
pnpm start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
